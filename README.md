# Desafio Pipeline ETL com Python 

Este repositório contém a resolução do desafio de projeto de ETL (Extract, Transform, Load) desenvolvido durante o bootcamp de Engenharia de Dados. O objetivo foi criar um pipeline robusto para manipulação de dados de clientes utilizando a linguagem Python.

## 📌 Contexto do Desafio
O objetivo foi simular um cenário real de Engenharia de Dados, onde realizamos as três etapas fundamentais de um pipeline de dados:
1. **Extract (Extração):** Leitura e extração de dados brutos de uma lista de usuários/clientes.
2. **Transform (Transformação):** Aplicação de regras de negócio personalizadas (como geração de mensagens automatizadas com base no perfil de cada cliente).
3. **Load (Carregamento):** Salvamento ou exibição dos dados já estruturados e enriquecidos.
